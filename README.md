# 🌤️ Weather Pro App

A responsive weather application built using HTML, CSS, and JavaScript.  
It fetches real-time weather data using the OpenWeather API and displays current weather, hourly forecast, and 5-day forecast.

---

## 🚀 Features
- Search weather by city name  
- Get weather using current location (GPS)  
- Real-time temperature, humidity, wind details  
- Hourly weather forecast  
- 5-day weather forecast  
- Dark mode toggle  
- Recent search history (local storage)  
- Dynamic weather-based background themes  
- Fully responsive design (mobile + desktop)

---

## 🛠️ Tech Stack
- HTML5  
- CSS3  
- JavaScript (Vanilla JS)  
- OpenWeather API
